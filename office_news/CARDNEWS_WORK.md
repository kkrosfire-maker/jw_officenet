## 생성 스크립트 요약

### `extract_post.py`
- 네이버 블로그 단일 포스트 텍스트·이미지 추출
- `PostView.naver` iframe URL로 직접 접근
- 출력: `output/wons0_0_224168110325/` 폴더

### `generate_mineoffice3.py` (최종)
- 실제 블로그 이미지 자동 합성 (`paste_photo()`)
- 고정 y 좌표 레이아웃 → 텍스트 overflow 방지
- 주요 함수:
  - `blog_img(n)` — n번 블로그 이미지 로드
  - `crop_center(img, w, h)` — 중앙 기준 크롭
  - `paste_photo(base, n, x, y, w, h, dark, radius)` — 어두운 오버레이 합성
  - `full_bg_photo(n, dark, tint)` — 전체 배경 이미지 처리
  - `put_wrap()` — 자동 줄바꿈 텍스트
  - `draw_logo()` — 하단 JW 로고
