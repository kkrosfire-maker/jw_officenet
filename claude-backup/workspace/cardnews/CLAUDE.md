# CLAUDE.md — 카드뉴스 디자인 가이드

## 캔버스 사이즈

| 구성 | 사이즈(px) | 특징 |
|------|-----------|------|
| 정방형 | 1080 x 1080 | 기본 피드용 |
| 세로형 | 1080 x 1350 | 노출 최적화 (기본 권장) |
| 스토리형 | 1080 x 1920 | 릴스/스토리에 최적화 |

## 폰트

### 제목용
- 지마켓 산스 (Gmarket Sans)
- 어그로체
- 에스코어 드림 (S-Core Dream)

### 본문용
- 프리텐다드 (Pretendard)
- 페이퍼로지 (Paperlogy)

### 추천 조합
- 프리텐다드 + 지마켓 산스 조합이 안정적

## 텍스트 크기

| 용도 | 크기 | 스타일 |
|------|------|--------|
| 표지 타이틀 | 60–80px | Bold |
| 부제목 | 45px | Regular |
| 본문 타이틀 | 45px | Bold |
| 본문 설명 | 28px | 얇게, 2~3줄 이내 |
| 출처 / 링크 | 25px | Bold |

## 색상

- Color Hunt에서 컬러 조합 참고
- 캔바 템플릿에서 색상 추출
- 부드러운 톤이나 무채색 계열 권장

## 구성

- 카드뉴스는 항상 **6장 이상**으로 구성 (5장 이하는 노출 약함)

### 기본 구성 순서
1. **표지 (후킹)** — 질문형 또는 흥미 유도 문구, 타이틀 80px 한 문장
2. **문제 제시**
3. **정보 A**
4. **정보 B**
5. **요약 정리**
6. **CTA** — 저장, 공유 등 행동 유도

## 작성 원칙

- 한 카드 = 한 메시지
- 본문 텍스트는 2~3줄 이내로 제한 (4줄 이상 시 독자 피로도 증가)
- 이미지 없이 텍스트만으로도 정보 밀도 높게 구성 가능
- 카드뉴스는 디자인보다 **구조와 기획**이 중요

## 기획 프롬프트 예시 (챗GPT 활용)

> "인스타그램 카드뉴스를 만들 거야. 주제는 '[주제]'이고, 6장 구성으로 정리해 줘. 첫 장은 질문형 훅, 마지막 장은 요약 + 저장 유도 문구."

채널 정보, 말투, 글자 수 제한을 함께 제공하면 더 효과적.
