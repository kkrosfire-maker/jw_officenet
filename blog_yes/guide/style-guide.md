# 박원종 내과 블로그 문체 스타일 가이드

> 이 가이드는 블로그 원문 7편을 분석하여 작성되었습니다.
> AI 에이전트가 이 파일 하나만 보고도 동일한 톤과 문체로 글을 쓸 수 있도록 구성했습니다.

---

## 1. 전체 톤 & 무드

- **친절한 동네 주치의** 느낌. 전문 의학 정보를 다루면서도 독자를 걱정하는 이웃 어른 같은 말투.
- 권위적이지 않고, 독자가 이미 알 것 같은 경험("분명 낮엔 따뜻해서 얇게 입고 나갔는데...")을 먼저 꺼내며 공감을 형성함.
- 겁을 주되 과하지 않게. 위기감을 조성하지만 마지막엔 반드시 "치료하면 낫는다"는 희망으로 마무리.
- 독자층은 **지역 주민(중장년층 포함)**, 전문 용어는 반드시 괄호로 쉽게 풀어줌.

---

## 2. 말투 & 경어체

### 기본 어미
- **합쇼체/해요체 혼용**: 본문 설명은 합니다/됩니다, 독자에게 말 걸 때는 ~하세요/~하시죠/~드릴게요
- 문장 끝 다양하게 변주:
  - `~입니다` / `~됩니다` (설명·사실 전달)
  - `~하세요` / `~마세요` (권고·조언)
  - `~하시죠?` / `~많으시죠?` (공감 유도)
  - `~드릴게요` / `~알려드립니다` (친절한 제공)
  - `~수 있습니다` / `~이어질 수 있어요` (경고·가능성)
  - `~하답니다` / `~쉬워요` (부드러운 정보 전달, 해요체)

### 주목할 어투 특징
- **"~죠?"** 로 독자 경험을 확인하는 질문을 자주 사용: "고생하시는 분들 많으시죠?", "생각하시죠?"
- **"~분들"** 호칭을 애용: "고생하시는 분들", "해당되시는 분들", "방치하시는 분들"
- 어르신 독자 고려 시 **"~하셔야 합니다", "~받으셔야"** 처럼 높임말 강화
- 직접 인용 형식으로 독자 심리 대변: `"아직 30대인데 벌써..."`, `"에이, 매년 이맘때면 이래~"`, `"좀 시린 것뿐인데..."`

## 3. 문장 호흡

- **단문과 중문을 교차**. 설명이 필요한 곳엔 쉼표로 이어가고, 강조할 땐 짧게 끊음.
- 한 문단 = 2~4문장 정도. 너무 길게 이어가지 않음.
- 긴 문장 뒤엔 짧은 결론 문장으로 마무리:
  > "결과적으로 말단 부위의 온도가 급격히 낮아지며 우리가 느끼는 '수족냉증'이 발생하는 것이죠."
- 단락 사이 여백(공백 줄)을 충분히 두어 가독성 확보.
- **"하지만", "결국", "특히", "바로"** 같은 접속·강조 부사로 흐름 전환.

## 4. 자주 쓰는 표현 & 즐겨 쓰는 어휘

### 경고/주의 환기 표현
| 표현 | 용도 |
|------|------|
| `~의 신호일 수 있습니다` | 증상을 질환과 연결할 때 |
| `~를 가볍게 보지 마세요` | 방치 경계 |
| `더 이상 참지 마세요` | 행동 촉구 |
| `절대 놓치지 마세요` | 골든타임 강조 |
| `~방치하면 ~으로 이어질 수 있습니다` | 합병증 경고 |
| `단순한 ~이 아닙니다` | 흔한 오해 교정 |

### 공감 형성 표현
| 표현 | 용도 |
|------|------|
| `~고생하시는 분들 많으시죠?` | 독자 경험 공감 |
| `흔히들 ~라고 생각하시죠?` | 일반적 오해 소개 |
| `나는 원래 몸이 차` / `이러다 말겠지` | 독자 심리 직접 인용 |
| `조금만 신경 쓰면` | 실천 장벽 낮추기 |

### 행동 촉구 표현
| 표현 | 용도 |
|------|------|
| `근처 병원에 들러서` / `근처 병의원을 방문해` | 내원 유도 (부담 없이) |
| `전문의와 상담해 보시기 바랍니다` | 정중한 권유 |
| `꼭 기억해 주세요` | 핵심 정보 강조 |
| `지금 바로` | 즉각 행동 유도 |

### 즐겨 쓰는 비유·은유
- 혈액 = **"난방수"** (열을 전달하는 역할)
- 당뇨 혈액 = **"설탕물"** (끈적함 표현)
- 몸 = **"24시간 풀가동되는 엔진"**
- 땀 = **"몸이 내뱉는 언어"**
- 혈관 문제 = **"침묵의 살인자"**, **"시한폭탄"**
- 조임쇠(괄약근) = **"문"** 으로 풀어 설명

### 강조 어휘
`치명적`, `골든타임`, `결정적`, `분기점`, `임계점`, `되돌릴 수 없는`, `가역성`, `가장 큰 원인`

## 5. 도입부 패턴

### 공식 인사 (거의 모든 글에 동일하게 등장)
```
안녕하세요! 여러분의 건강 주치의 박원종 내과입니다. 😊
```
- 이모지 😊 는 인사말 끝에 항상 붙음.
- 일부 글에서는 "든든한 건강 지킴이"로 변주.

### 도입 전개 3단계
1. **계절·생활 공감**: 요즘 날씨나 연휴 같은 시의성 있는 상황 묘사로 시작
   > "갑작스러운 한파 소식에 몸이 절로 움츠러드는 요즘입니다."
   > "요즘 아침저녁으로 공기가 갑자기 쌀쌀해졌죠?"

2. **독자 오해 또는 방치 행동 짚기**: 독자가 흔히 넘기는 증상/행동을 꺼냄
   > "단순히 체질 탓이라 생각하며 '다한증'이겠지, 라며 가볍게 넘기셨나요?"

3. **오늘 글의 목적 한 줄 선언**:
   > "오늘은 ~에 대해 이야기해 보려고 합니다!"
   > "오늘은 ~을 요약해 드립니다."

---

## 6. 본문 구성 패턴

- **소제목 + 번호**: `1. 제목`, `2. 제목` 형식으로 섹션 구분
- 소제목에 **이모지** 적극 활용: 🍗 🚨 👨‍⚕️ 💡 🌡️ ⚠️ 🧣 🔍 🩺
- 각 섹션은 **원인 설명 → 증상 나열 → 권고 행동** 흐름
- 증상·항목 나열 시 **불릿 리스트** 사용, 각 항목의 **핵심 키워드는 볼드** 처리
- 전문 용어는 괄호로 즉시 풀어줌: `조임쇠(괄약근)`, `경동맥 초음파 (혈관의 두께 확인)`
- 체크리스트 형식 활용: `- [ ] 더위를 남들보다 유난히 더 못 참게 되었다.`

## 7. 마무리 패턴

### 핵심 메시지 인용구 (명언체)
거의 모든 글에 볼드 또는 따옴표로 한 문장 요약:
> `"오늘의 검진 예약이 10년 뒤 당신의 생명을 결정합니다."`
> `"땀은 우리 몸이 내뱉는 언어입니다"`
> `"적당히 먹고, 식후 3시간은 눕지 마세요."`

### 행동 촉구 마무리
- 내원을 부드럽게 권유하는 문장으로 마무리: "근처 병의원에 편안하게 걸음 하셔서..."
- "참지 마세요", "미루지 마시고" 표현으로 지연 방지

### 법적 고지 (고정 문구)
```
※ 모든 치료 및 예방접종은 개인의 상태에 따라 발열, 통증, 알레르기 반응 등의 부작용이 나타날 수 있으므로, 반드시 의료진과 충분한 상담 후 진행하시기 바랍니다.
```

### 마지막 인사 (고정 문구)
```
박원종 내과에서 전해드리는 건강 정보였습니다. 오늘도 건강하고 평안한 하루 보내세요!
```

## 8. 기타 눈에 띄는 특징

### 이모지 사용 규칙
- 소제목 앞에 주제와 관련된 이모지 1개
- 인사말 끝에 😊 고정
- 본문 내 강조 이모지: ⚠️(경고), 💡(팁/요약), 🔍(검진), 🩺(진료)
- 과하게 쓰지 않음. 섹션당 1개 수준 유지

### 독자 직접 호명
"여러분", "어르신들", "30대라면", "기저질환자" 등 독자를 세분화하여 직접 지칭

### 수치·데이터 활용
신뢰도를 높이기 위해 구체적 수치 삽입:
> "기온이 10도 떨어지면 수축기 혈압은 약 10mmHg 상승"
> "겨울철 사망률이 여름보다 23%나 높은"
> "식도 궤양으로 진행 확률 70% 이상"

### 의성어·구어 표현
딱딱한 의학 정보를 부드럽게 만들기 위해 구어체 단어 사용:
`으슬으슬`, `찌릿찌릿`, `간질간질`, `줄줄`, `뚝`, `꽉`, `딱`, `꼼꼼하게`

### 강조 문장 처리
- 중요 문장 전체를 **볼드** 처리
- 특히 경고성 내용, 행동 촉구 문장에 적극 활용
- 핵심 단어만 볼드 처리하는 방식도 혼용
