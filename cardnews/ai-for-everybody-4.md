## 1. 툴 선택과 작업 준비

### 어떤 툴을 쓸까?

- **캔바**: 카드뉴스 초보자 추천. 템플릿이 풍부하고 디자인 선택지가 많아 디자인 고민을 줄일 수 있음
- **피그마**: 텍스트 간격, 정렬, 이미지 배치 등을 세밀하게 조정할 수 있으며, 카드별 일관성 유지에 유리

### 카드뉴스 기본 사이즈

| 구성 | 사이즈(px) | 특징 |
|------|-----------|------|
| 정방형 | 1080 x 1080 | 기본 피드용 |
| 세로형 | 1080 x 1350 | 노출 최적화 |
| 스토리형 | 1080 x 1920 | 릴스/스토리에 최적화 |

세로형(1080x1350)이 인스타그램 노출에 가장 유리함.

### 폰트와 색상 선택

#### 폰트 추천

| 용도 | 폰트 |
|------|------|
| 제목 | 지마켓 산스, 어그로체, 에스코어 드림 |
| 본문 | 프리텐다드, 페이퍼로지 |

#### 텍스트 크기

| 항목 | 크기(px) | 스타일 |
|------|---------|--------|
| 표지 타이틀 | 60~80 | Bold |
| 부제목 | 45 | Regular |
| 출처/링크 | 25 | Bold |
| 본문 타이틀 | 45 | Bold |
| 본문 설명 | 28 | 얇게, 2~3줄 이내 |

#### 색상 선택
- Color Hunt에서 컬러 조합 찾기
- 캔바 템플릿에서 색상 추출
- 부드러운 톤이나 무채색 계열 추천

## 2. 카드뉴스 기획 구성

### 카드 수는 몇 장이 적당할까?

최소 6장 이상 권장. 5장 이하는 노출이 약한 경향.

#### 기본 구성
1. **후킹**: 질문 또는 흥미 유도
2. **문제 제시**
3. **정보 A**
4. **정보 B**
5. **요약정리**
6. **CTA**: 저장, 공유 등 행동 유도

### 챗GPT로 카드뉴스 기획하기

프롬프트 예시:
> "인스타그램 카드 뉴스를 만들 거야. 주제는 '챗GPT 프롬프트 작성 팁'이고, 6장 구성으로 정리해 줘. 첫 장은 질문형 훅, 마지막 장은 요약 + 저장 유도 문구."

채널 정보, 말투, 글자 수 제한 등을 미리 제공하면 더 효과적.

## 3. 직접 만들어본 결과와 팁

### 자주 쓰는 구성법
- 표지 타이틀: 80px, 한 문장 Hook
- 본문: 45px 타이틀 + 28px 설명 2줄
- 프리텐다드 + 지마켓 산스 조합이 안정적
- 이미지 없이 텍스트만으로도 정보 밀도 높게 구성

### 실수했던 부분
- 색상 대비 및 폰트 사이즈가 너무 작아 가독성 저하
- 텍스트가 4줄 넘으면 독자 피로도 증가 → 2~3줄 제한
- 한 장에 너무 많은 정보 → "한 카드 = 한 메시지" 원칙 준수
- 지인 유저 테스트를 통해 개선
